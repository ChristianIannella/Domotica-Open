# SPECIFICHE TECNICHE

- Alimentazione 5VDC
- MCU ESP8266
- WiFi
- 1 GPIO (D14)
  


| PINOUT |  |
|---|---|
|Service Button|D12 (10K external pull-up)|
|Switch Button|D14 (10K external pull-up)|
|Switch Led|D13 (10K external pull-up)|
