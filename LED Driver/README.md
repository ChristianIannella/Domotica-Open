# SPECIFICHE TECNICHE

- Alimentazione 5VDC
- MCU ESP8266
- WiFi
- 1 GPIO (D13)
  


| PINOUT |  |
|---|---|
|Service Button|D12 (10K external pull-up)|
|Strip Led Output|D13|
